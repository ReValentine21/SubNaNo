# 🐦 SubNaNo Flap

> **Flappy Bird на Arduino Nano + OLED 128×64**  
> Одна кнопка. Один дисплей. Бесконечное страдание.

## 🔧 Железо

| Компонент | Подключение |
|-----------|-------------|
| Arduino Nano | — |
| OLED SSD1306 128×64 | I²C → A4 (SDA), A5 (SCL) |
| Кнопка | D2 → GND (с подтяжкой INPUT_PULLUP) |

> **Библиотеки:** `Adafruit_GFX`, `Adafruit_SSD1306`

---

## 🎮 Управление

| Экран | Действие | Результат |
|-------|----------|-----------|
| Меню | Нажать | Старт игры |
| Игра | Нажать | Прыжок |
| Game Over | 1 клик | Меню |
| Game Over | 2 клика | Рестарт |

---

## ⚡ Установка

1. Клонировать репозиторий:
```bash
git clone https://github.com/ReValentine21/subno-flap.git
```

2. Открыть `subno_flap.ino` в Arduino IDE

3. Установить библиотеки через **Tools → Manage Libraries**:
   - `Adafruit SSD1306`
   - `Adafruit GFX Library`

4. Выбрать **Board: Arduino Nano**, нужный COM-порт

5. Загрузить — и летим 🐦
